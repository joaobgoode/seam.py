# SEAM PY

Seam carving algorithm in Python, its very slow, made for educational purposes.

Do not use this code for anything serious.

Also provides grayscale and sobel filters.

## Usage

### Grayscale

```console
grayscale.py input.png output.png
```

![grayscale](https://github.com/joaobgoode/seam.py/blob/main/example/grayscale.png)

### Sobel

```console
sobel.py input.png output.png
```

![sobel](https://github.com/joaobgoode/seam.py/blob/main/example/sobel.png)

### Seam Carving

```console
seam.py input.png 500 output.png
```

500 is the amount of seams to remove.

![seam](https://github.com/joaobgoode/seam.py/blob/main/example/carved.png)
